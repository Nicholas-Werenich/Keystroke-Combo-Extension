# keystrokeComboRad README

A way to make using VS Code more fun, while typing wrack up combos and points to add to your total score.

## Features

Default is 5 points per character and 10 points per new line.
Opening a new pair of brackets (any kind) adds a 0.5 to your multiplyer.
Typing or deleting characters resets your timer, when the timer runs out your multiplyer is reset and your combo is added to your score.

![Panel View\]\(assets/Demo.gif\)

Pop ups for adding points and breaking combo.
Inline effects while typing.

## Extension Settings

Togglable features:
Notifications
Progress bar
Inline effects

Can change timer, point values and multiplyer.
